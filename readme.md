Student friendly linear optimisations solver supporting different algorithmic variants. 

## How to setup Simpler?
1. Download release based on your OS. Offered distributions are Windows, Mac and Linux. Java (.jar) support is under construction.
2. Make sure port 8080 is free
3. Run downloaded binary

### Windows specific instructions
- Download .exe and start it up with double click
- It might be neccessary to explicitly allow it to run in a system prompt window since it's downloaded from the internet. 

### Mac specific instructions
Mac instructions show application running from downloads directory. If you move it somewhere else, adjust your commands accordingly.
- Download release (tutorial will use instance Simpler_0.1.0-aplha.1_macos as an example)
- Open up our terminal and cd into your downloads directory
```zsh
cd $HOME/Downloads
```
- Make application executable
```zsh
chmod +x ./Simpler_0.1.0-aplha.1_macos
```
- Run application
```zsh
./Simpler_0.1.0-aplha.1_macos
```
- Mac doesn't usually allow running internet downloaded binaries like this. In case of OS not allowing you to run Simpler, visit Settings/Privacy&Security and in Security section, you should be able to allow Simpler to run.
- Run again in terminal and click all requested allows
```zsh
./Simpler_0.1.0-aplha.1_macos
```
- To exit the application use CONTROL + C in terminal
